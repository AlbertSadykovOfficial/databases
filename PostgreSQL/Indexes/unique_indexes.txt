

	Уникальные индексы

			Индекы могут быть использованы для обеспечения уникальности значений.
			В таком случае мы уже не сможем ввести в строки одинаковые имена, но 
			при этом сможем вводить NULL-значения, потому что каждое отдельное 
			NULL-значение считается уникальным.

			Если уникальный индекс создан по нескольким атрибутам, то совпадением
			считается комбинация значений в 2х строках.

			Создать:

					CREATE UNIQUE INDEX <имя-индекса> ON <имя таблицы> (имя столбца, ...);
					CREATE UNIQUE INDEX aircraft_unique_model_key ON aircrafts ( model );
