

		Индексы по нескольким столбцами.

				Индексы могут создаваться по нескольким столбцам, что повысит скорость.
				Так же индексы желательно использовать при низкой селективности (выборке значений).

				Сравнение:

						Установим индекс:

								CREATE INDEX tickets_book_ref_test_key ON tickets ( book_ref );

						Выполним запрос:

								SELECT * 
									FROM tickets
								OREDER By book_ref
								 LIMIT 5;

						(УДАЛИМ индекс и выполним запрос вновь -> время увеличится)
