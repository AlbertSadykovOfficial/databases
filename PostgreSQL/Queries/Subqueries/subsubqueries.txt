

		Помимо обычных подзапросов могут встречаться и вложеные подзапросы (один внутри другого).
				
				Такие запросы выгляят следующим образом:
					
					SELECT
						FROM (SELECT
							...
							(SELECT
								...
							)
						)


				Чтобы сделать запрос более наглядным и интуитинвным можно использовать:
				Общее табличное выражение (CTE) - WITH ts AS ( ... ), чтобы выделить такие фрагменты.

					WITH ts AS
					(	SELECT
						(SELECT
							...
						)
					)
					SELECT 
							...
						FROM ts
