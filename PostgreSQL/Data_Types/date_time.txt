
		
		Типы дата/время

				Даты - в соответствие с Григорианским календарем

				Моментальные значения:
						SELECT current_(VAR); # VAR - date, time, timestamp
												
				Получение даты и времени осуществляется через приведение типов (::):

						SELECT '2021-03-11'::date
						SELECT 'Mar 11, 2021'::date

						SELECT '23:12:45 '::time
						SELECT '11:12:45 pm'::time

				Временная отметка - интегральный тип, включающий дату и время

						Существуют 2 типа: с отметкой времени и без, но занимают они одинаковое
						кол-во места, так как обычный timestamp хранит отметку +0, 
						а при извлечении и выводе резульатата он ее удаляет.  
						SELECT timestamp '2021-03-11 23:12:45'
						SELECT timestampz '2021-03-11 23:12:45'
						SELECT timestamp with time zone '2021-11-03 23:12:45'

						Вывести дату в другом формате:
							SELECT to_char( current_date, 'dd-mm-yyyy' );

				Интервал - продожительность отрезка времени

						Поскольку это отрезок между 2 временными отметками, то значение этого
						типа можно получить при вычитаии одной временной отметки из другой

						SELECT '1 year 2 month ago'::interval
						SELECT 'P0001-02-03T04:05:06'::interval

						SELECT  ('2021-03-11'::timestamp - '2021-03-01'::timestamp )::interval


				Усечение до определенной точности:

						SELECT ( date_trunc( 'hour', current_timestamp ));

				Извлечение отдельных полей:

						SELECT extract( 'mon' FROM timestamp '1999-05-13 23:12:45.12345')
