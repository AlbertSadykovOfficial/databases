
		Строковые типы

			character varying(n) - Динамическая длина символов строки, не выше заданной
			character(n) - фиксирванная длина, если строка имеет меньше n символов, она дополняется пробелами
			text - для больших текстов

			или по всевдонимам:
			varchar(n)
			char(n)

			Пример:
				SELECT 'PostgreSQL'

			Чтобы включить в состав запроса спец символы, есть 2 метода:
				Удвоение : SELECT 'PGDATY''17'
				Обособление : SELECT $$PGDATY'17$$  (такой стиль ункикален для Postrgre, его нет в стандартном SQL)

			Расширение - строковые константы в виде языка C

					При использовании данного расширения все, что пишется внутри кавычек, имеет синтаксис C,
					отсюда и использование (\) для предворения спец символов:
							
							SELECT E'PGDAY\n17';
							SELECT E'PGDAY\'17';
