

		Вывод данных из файла:

				Вставим данные из файла (добавим).
				Учтем, что разделитель данных - табуляция,
				конец файла заканчивается 2мя символами - "\."

				COPY aircrafts_tmp FROM '/home/postgres/aircrafts.txt'

				При выполнении команды COPY выполняются все проверки ограничений,
				наложенных на таблицу, поэтому ввести дублирующие данные не получится.

		Ввод данных в файл:

				Выведем данные в файл в формате csv:
				(Если не указать формат, то стандартный разделитель - табуляция)
				
				COPY airctafts_tmp TO '/home/postgres/aircrafts_tmp.txt' 
					WITH ( FROMAT csv );
