

		Схемы базы данных


				Схема - логический фрагмент БД в котором могут содержаться различные объекты:
								(Таблицы, Представления, Индексы и др), имена этих объектов должны 
								быть уникальны в пределах этого фрагмента.

				Посмотреть список схем в БД:

						\d

				Переключиться на схему:

						SET search_path = bookings

				Если схема не выюрана, то обращение к конкретным объектам БД осуществляется так:

						SELECT * FROM <schema-name>.<table-name>

				Создать обхект в схеме:

						CREATE TABLE <schema-name>.<table-name>
							...
